PART A: DDL (Data Definition Language)
1. Create Database

Create a database named shop_db.

=> create database shop_db;
	
2. Create Table: products

Create a table with the following structure:

Field Name	Data Type	Description
product_id	INT	Primary Key, Auto Increment
product_name	VARCHAR(100)	Name of the product
category	VARCHAR(50)	Product category
brand	VARCHAR(50)	Brand name
price	DECIMAL(10,2)	Product price
quantity	INT	Available stock
status	VARCHAR(20)	Available / Out of Stock
created_at	DATE	Date added

ðŸ“Œ Constraints:

product_id â†’ Primary Key

price must be greater than 0

quantity must not be negative

=> 
use shop_db;

create table products(
    product_id int PRIMARY key AUTO_INCREMENT,
    product_name varchar(20) ,
    category varchar(30),
    brand_name varchar(30),
    price decimal(10,2) CHECK (price>0),
    quantity int check (qunatity>0),
    status varchar(20),
    create_at date 
 );


3. Alter Table

Add a column discount (DECIMAL(5,2))

=> alter table products add column discount decimal(5,2) ;

Rename column status to availability

=> alter table products change STATUS availability varchar(20);

Drop the column discount

=> alter table products drop column discount;

4. Drop Table

Write a query to drop the products table.

=> drop table products;

6. Update Records

Update the price of a product by product_id

=> update products set price = 4000 where product_id =3;

Increase quantity by 10 for all products in Electronics category

=> update products set quantity = quantity+10 where category="electronic";

Change availability to Out of Stock where quantity = 0

=> update products set availability = "out of stock" where quantity = 0;

7. Delete Records

Delete a product by product_id

=> delete from products where product_id=1;

Delete all products where availability = Out of Stock

=> delete from products where availability = "out of stock";

PART C: SELECT Queries
Basic SELECT

Display all products

=> SELECT * FROM `products`;

Display product name, price, and category only

=> SELECT product_name,price,category FROM `products`;

WHERE Clause

Display products with price greater than 500

=> select * from products where price >500;

Display products belonging to category Electronics

=> SELECT * FROM `products` WHERE category = "electronic";

Display products with quantity between 10 and 50

=> SELECT * FROM `products` WHERE quantity BETWEEN  10 and 50;

ORDER BY Clause

Display all products ordered by price ascending

=> SELECT * FROM `products` order by price asc;

Display all products ordered by quantity descending

=> SELECT * FROM `products` order by quantity desc;

LIKE Clause

Display products whose name starts with 'S'

=> SELECT * FROM `products` WHERE product_name like "s%";

Display products whose brand contains 'tech'

=> SELECT * FROM `products` WHERE product_name like "%tech%";

Display products whose name ends with 'Pro'

=> SELECT * FROM `products` WHERE product_name like "%pro";


LIMIT and OFFSET

Display the first 5 products

=> SELECT * FROM `products` limit 5;

Display 5 products starting from the 6th record

=> SELECT * FROM `products` limit 5 OFFSET 5;

Display the top 3 most expensive products

==> select * from products order by price desc limit 3;

GROUP BY Clause

Display total number of products in each category

=> select category , sum(quantity) from products group by category;

Display average price of products by brand

=> select brand_name , avg(price) from products group by brand_name;

Display total stock (sum of quantity) per category

=> select category , sum(quantity) from products group by category;

HAVING Clause

Display categories having more than 3 products

=> select category , sum(quantity) from products group by category having sum(quantity)>3;

Display brands where average price is greater than 1000

=> select brand_name , avg(price) from products group by brand_name having avg(price) > 1000; 

PART D: Combined Queries

Display category-wise total products where total products > 2, ordered by total products descending

=> SELECT category , count(*) from products group by category having count(*) > 2;

Display brands with total stock less than 50

=> select brand_name , sum(quantity) from products group by brand_name having sum(quantity) < 50;

Display the most expensive product in each category

=> select category , max(price) from products group by category;
